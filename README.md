# rna-seq-len-est-0

Estimate a transcript's length from RNA-Seq data

## Usage

This program uses [BLAT](http://genome.ucsc.edu/FAQ/FAQblat.html) to align reads to a reference genome, 
and estimate a transcript's start and end site.
